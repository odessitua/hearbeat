# Подготовка к Digital Future Hackathon

Рабочая папка команды **HearBeat** — голосовые чек-ины для семей на расстоянии (ниша **Aging & Longevity**).

**Хакатон:** [Digital Future Hackathon](https://hackathon.lezo.io/) · 11–12 июля 2026 · онлайн, AI-first / no-code

**Стратегия команды:** «полная продуктовая модель + минимальное демо». На сцене — целевой продукт; на экране — доказательство AI-core за выходные.

## HearBeat в одном абзаце

**HearBeat** — сервис заботливых голосовых чек-инов для семей на расстоянии. У пожилого родителя — простой сценарий «входящий звонок»; после короткой беседы сервер анализирует голос и смысл. Взрослый ребёнок видит тренд самочувствия и сигнал «сегодня лучше позвонить самому».

**AI-core (демо):** акустика + персональный baseline → `vitality_score` + семантическое summary.

**Vision (не в демо, но в презентации):** адаптивный «умный звонок», family prompt, микро-утилиты в диалоге.

## Критерий успеха MVP (выходные)

Один публичный сценарий без пояснений разработчика:

```text
Открыть ссылку → ответить на чек-ин → увидеть обновлённый dashboard →
понять, почему система советует «стоит позвонить».
```

**Обязательные артефакты сдачи:** питчдек PDF (≤5 слайдов), OnePager JPG 2400×3500, публичная ссылка на демо (+ Loom как запасной вариант).

## Три слоя продукта (не смешивать на питче)

| Слой | Что это | Приоритет на хакатоне |
|---|---|---|
| **1. AI-core** | Акустика + baseline + тренд + сигнал семье | **Демо** |
| **2. «Умный звонок»** | Адаптивная беседа, family prompt, календарь | **Vision / Phase 1.5** |
| **3. Микро-утилита в разговоре** | Рецепт, совет, тема про внука — внутри чек-ина | **Vision** |

**Формула питча:** «Вот как HearBeat выглядит целиком → вот что собрали за выходные → вот следующие шаги.»
